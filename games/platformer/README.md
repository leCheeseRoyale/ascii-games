# Platformer Template

Side-scrolling platformer with gravity, jumping, and collectible stars.

## API

`defineScene` + ECS (real-time).

## What It Demonstrates

- Gravity and jumping physics
- Platform collision system (one-way platforms)
- Collectible items with a collection system
- Entity factories for player, platforms, and stars
- Star spawning system for procedural placement
- Score tracking via the React UI overlay

## Who Should Use This

Developers building side-scrollers, runners, or any game with gravity and platforming. Shows how to layer custom physics (platform collisions) on top of the built-in physics system.

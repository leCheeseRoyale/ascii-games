# Asteroid Field Template

Top-down space shooter. Dodge and destroy asteroids with your ship.

## API

`defineScene` + ECS (real-time).

## What It Demonstrates

- Full game loop: title, play, and game-over scenes
- Entity factories for player, bullets, and asteroids
- Custom systems: player input, collision detection, asteroid spawning
- Shooting mechanics with velocity-based projectiles
- Score and health HUD via the React UI overlay
- Screen wrapping and entity lifecycle

## Who Should Use This

Developers building real-time action games (shooters, bullet-hell, space games). Good intermediate reference for how entities, systems, and scenes fit together in a complete game.
